This is a solution for when you start the laravel server (php artisan serve) and make changes 
then save but the changes are not reflected on the website. You have to force restart to fix it.

* Edit PHP version file, use this command to find your php file
$ php --ini | grep "Loaded Configuration"

* Use your file path here to change
$ sudo nano /etc/php/php.ini 

* You have to change these two values to 0 and save
[opcache]
opcache.enable=0
opcache.enable_cli=0

* This will disable OPcache globally which is good for production but not development.
(I tried .user.ini but it does not work)

* Now simply restart the laravel server.
